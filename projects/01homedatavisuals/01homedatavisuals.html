Home Data Visualization

Visualization of home gathered data

In the Home Data project we developed a network sensors for transformable housing enviroments. Once the data was collected, we decided to build a visual tool for the first analysis and to make it easier for the user to see his own behavior! Using Dash from plotly a dashboard building tool using python.

The idea is to build an easily customizable tool, where graphs and tables are easy to change and adapt to what the users understands better. A first demo can be found in this github repo .

At the moment, the build features are:

  • Project description window, with simple information about the project

  • Historical data analysis window. Here the user can find graphs analysinzg his interaction with different moving kitchen components, or the relationship between enviromental variables like sound, light, humidity or temperature in a time scale. Besides, with a data table is easy to go trough the day and see the data for every moment

  • Storage window. As we are working in a kitchen data project, and storage is key, here we included the measure of main enviromental variables for storing food like temperature, humidty and light. Also graphs aiming to show the enviromental status of the whole space.
