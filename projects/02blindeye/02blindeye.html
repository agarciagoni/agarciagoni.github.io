BlindEye

Human activity recognition through computer vision

The aim of this project is to develop an activity recognition algorithm with different applications ranging from elderly care, anomaly and alarm detection to architectural robotics or learning gamification... In a privacy preserving way! More info about the complete proyect here! .

The project focus on three different main blocks.


  • Human activity recognition (HAR) software. For the first iteration we are using YOLO for object detection and classification along with OpenPose for pose estimation. Besides depth is analyzed to estabilish interactions between people and objects.

  • Product design. The goal is to design a camera that everyone will feel comfortable having in their house, and design also plays an important role for user acceptance. Shouldn't look like a surveillance camera, should be appealing, like a piece of modern art and should be easily turnned off every time the user want to.

  • P R I V A C Y. Cameras inside my house? No way. The project aims to design a system where the video is not accesible from outside, where the only output is the activity detection that is only shared with whoever the user defines (a concerned family member for example) or it just stays inside the house and is used to maximize the housing experience with enviroment control, transformable spaces and other features among the architectural rotobitcs.
