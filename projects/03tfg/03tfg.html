Electric Machines Software

Axial Flux Syncronous Machines Fast Calculus Software

One of the main problems when designing electric motors is the time comusing 2D and 3D simulations used to validate designs before manufacturing the first prototypes. To make this stage agile, this project aimed to develop a software that will calculate the parameters and performance of an axial flux permanent magnet syncronous machine in a fraction of the tame it will take in comercial simulation software with enough accuracy.

Working with the research center Ceit - IK4 we produce a software with the desired results coding it in Matlab and validated with ANSYS Maxell:

Our software takes 1minute and 3 seconds while the comercial software takes between 3 to 5 days, getting an error of 2-5% !

This accuracy and the small computation time makes it a usefull tool for a first iteration calculus, then selecting the most promising one for the accurate 3D simulation.
